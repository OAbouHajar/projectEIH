    Emergency Info Hub

    Emergency Info Hub(EIH), Is a central website helps the emergency services to prepare for, respond to & recover from disaster, by providing all needed data for the targeted building (E.g. Number of people, area size and emergency exits).
    The main objective of this project is to give the number of people inside the buildings using a simple sensors reading and face detection technology (where this project has been inp has never been provided by any emergency systemes before).
